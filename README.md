# Financial Document Question Answering


A **Streamlit-based web application** that allows users to upload financial documents (PDFs or Excel files) and ask questions to extract relevant information. The app uses AI-based techniques for document parsing and question answering.

## 🔗 GitHub Repository
[https://github.com/suryavamsi20/Financial-qa](https://github.com/suryavamsi20/Financial-qa)

---

## 🛠 Features
- Upload PDF or Excel financial documents (up to 200MB per file)
- Ask questions related to the uploaded financial document
- Extract answers and insights from financial statements automatically
- Interactive and user-friendly interface
